## Overview
This is my first data analysis project exploring liquor sales data from Iowa.

## Project Goals
- Clean and prepare the dataset for analysis
- Identify top-selling cities and products
- Explore sales trends over time
- Practice data visualization and analytical thinking

## Key Findings
- Des Moines has the highest total liquor sales by dollar amount
- "Juarez Gold DSS" is the top-selling product
- Sales peaked in 2016, potentially due to regulatory changes
- Significant sales decline in 2021-2022, likely pandemic-related

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
